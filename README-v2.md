# WAREHOUSE 8v8 — 웹 브라우저 FPS (v2)

최대 16명(팀당 8명) + AI 봇으로 즐기는 웹 FPS. 맵 2종, 고품질 절차적 3D 에셋, 칼·수류탄 포함. 서버 없이 GitHub Pages(정적 호스팅)에서 동작합니다.

> 이 파일은 신버전입니다. 기존 `index.html`(구버전)은 그대로 보존됩니다.

## 파일

- `warehouse-v2.html` — 신버전 게임. 배포 시 이 파일을 `index.html`로 올리면 됩니다.
- `intro.mp4` — 첫 로딩 때 재생되는 인트로 영상. **index.html과 같은 위치(저장소 루트)에 함께 올려야** 합니다.
- 배포용 zip(`warehouse-v2-dist.zip`)에는 `index.html`(게임)과 `intro.mp4`가 함께 들어 있습니다. 압축 풀어서 두 파일을 같이 올리면 됩니다.

## 새로워진 점 (v2)

- **맵 2종 선택**: 방장이 대기실에서 고름
  - 웨어하우스: 실내 컨테이너 전장
  - 필드(대형): 숲·갈대·건물·바위·컨테이너 야적장이 있는 야외 대형 맵 (배틀그라운드 스타일)
- **고품질 에셋**: 팔다리·헬멧·전술조끼를 갖춘 캐릭터(걷기 모션), 디테일한 총기, 주름진 컨테이너, 나무/바위/건물
- **팀 의상색**: 레드팀 검정, 블루팀 네이비
- **수류탄 개편**: 5번 무기로 이동. 좌클릭을 꾹 눌러 게이지를 채워 던지며(오래 누를수록 멀리), 폭발 반경 확대, 반경 안은 즉사
- **스나이퍼 상향**: 명중하면 거리와 부위에 상관없이 한 방에 즉사 (단, 무줌은 탄퍼짐이 크니 우클릭 줌 권장)
- **인트로 영상**: 페이지 첫 로딩 때 `intro.mp4` 재생(우상단 '건너뛰기'로 스킵). 브라우저가 소리 자동재생을 막으면 화면을 한 번 탭하면 소리가 나옵니다

## GitHub Pages 배포

1. 새 저장소 생성
2. zip을 풀어 `index.html`과 `intro.mp4`를 **둘 다** 저장소 루트에 업로드 (영상이 없어도 게임은 동작하며, 인트로만 건너뜁니다)
3. Settings → Pages → Branch `main`, 폴더 `/ (root)` → Save
4. 1~2분 후 `https://<아이디>.github.io/<저장소>/` 접속

## 멀티플레이 (P2P)

방 만들기 → 방 코드 공유 → 참가하기. 팀당 최대 8명, 빈 자리는 AI가 채웁니다. 게임 중 나간 사람 자리는 AI로 자동 전환.

## 조작

| 키 | 동작 |
|---|---|
| W A S D | 이동 |
| Space | 점프 |
| 좌클릭 | 발사 / (수류탄 선택 시) 꾹 눌러 차징 후 떼서 투척 |
| 우클릭 | 스나이퍼 줌 전환 (2배 → 4배 → 해제) |
| 1 / 2 / 3 / 5 | 라이플 / 스나이퍼 / 칼 / 수류탄 |
| R | 재장전 |
| Enter | 채팅 |
| Tab | 점수판 |

## 규칙

- 목표 킬(30/50/100, 방장 선택)을 5분 안에 먼저 달성하는 팀 승리
- 부활 후 5초 무적 / 적 체력바 비표시·벽 뒤 적 이름표 가림 / 구호상자(+40) 수시 등장

## 최근 개선 (v2.1)

- **연결 보강(TURN)**: 무료 TURN 서버(Open Relay)를 추가해 회사 방화벽·대칭 NAT 뒤에서도 접속 성공률을 높였습니다. (무료 서버라 가끔 불안정할 수 있고, 자주 쓰면 자체 TURN 권장)
- **스나이퍼 명중 수정**: 반동이 탄 방향보다 먼저 적용돼 매 발이 위로 솟던 버그를 고쳤습니다. 이제 조준한 곳으로 정확히 나갑니다. 추가로 멀티 지연 보정용 관용 판정(적을 향해 쏘면 최대 1km까지, 벽이 막으면 무효)을 넣었습니다.
- **줌 토글**: 우클릭을 누르고 있는 방식이 아니라, 누를 때마다 2배 → 4배 → 해제로 전환됩니다.
- **건물 창문**: 건물 벽에 작은 창을 내서, 안에 숨어 요새처럼 쓰며 창으로 사격할 수 있습니다.
- **적 이름표 제거**: 적의 이름표·체력바를 일절 표시하지 않습니다(아군만 표시). 갈대밭·건물·어둠을 이용한 은엄폐 전략이 가능합니다. 적·아군 구분은 의상색(레드=검정/블루=네이비)으로.
- **인트로**: 중간에 끊기지 않고 끝까지 재생(건너뛰기 누를 때만 스킵). 소리 자동재생을 브라우저가 막으면 화면을 한 번 클릭하면 소리가 켜집니다.
